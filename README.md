# Florida Fertilizer Certification Trainer v2

Green Industry Academy™ prototype.

## Contents
- `index.html`
- `styles.css`
- `app.js`

## Features
- Six study modules
- Practice mode with explanations
- 20-question randomized exam simulator
- Field scenario trainer
- Local browser progress tracking
- Mobile-friendly static design

## Deploy to GitHub + Netlify

1. Create a new GitHub repository.
2. Upload these files to the repository root.
3. In Netlify, choose **Add new site → Import an existing project**.
4. Connect GitHub.
5. Select the repository.
6. Build command: leave blank.
7. Publish directory: `/` or leave as repository root.
8. Deploy.
